How I work

My current design workflow

I am still learning UI design, so my workflow is simple and course-based. I start by understanding the goal of the page, then I plan the structure, choose a consistent visual style, build the interface, and review it for clarity.

1

Understand the goal

Identify the message of the page and what the user should notice first.

2

Plan the structure

Organize information into clear sections using hierarchy and grouping.

3

Build with consistency

Use Bootstrap, repeated spacing, repeated card styles, and a limited palette.

4

Review and improve

Check readability, contrast, balance, and whether each visual choice supports the content.
